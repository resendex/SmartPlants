## Instruções

### Como executar o protótipo

1. Abrir o ficheiro `html/home.html` num browser (Chrome, Firefox, Safari, Edge)
2. Navegar pelo protótipo utilizando os menus e botões disponíveis

### Estrutura do projeto

- `html/` - Páginas HTML do protótipo
- `css/` - Ficheiros de estilos CSS
- `js/` - Scripts JavaScript
- `multimedia/` - Imagens e outros recursos multimédia

### Notas

- O protótipo utiliza `localStorage` para persistir dados localmente no browser
- Para uma experiência completa, recomenda-se criar uma conta e fazer login
- As funcionalidades principais incluem: gestão de plantas, sistema de rega, notificações, calendário e fórum
